Crystal Game
Version 1.0 9/8/18

https://mrblrzl.github.io/unit-4-game/

There are four crystals displayed as buttons on the page.
A random number will be displayed at the start of the game.
When the player clicks on a crystal it will add a specific amount of points to the total score.
If the total score matches the random number, the player wins.
If the total score goes above the random number, the player losses.
The number of games won and lost is displayed.
